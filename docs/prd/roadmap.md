## Fase 1: MVP (Mês 1-2)

### Objetivo
Criar uma prova de conceito funcional que valide engajamento infantil e interesse dos pais.

### Entregas

| Semana | Entrega | Status |
|--------|---------|--------|
| 1 | Definição do personagem guia | Pendente |
| 1 | Design system infantil | Pendente |
| 2 | Estrutura de trilha (5 aulas) | Pendente |
| 2 | Conteúdo tema 1: Animais | Pendente |
| 3 | Desenvolvimento frontend | Pendente |
| 3 | Sistema de áudio | Pendente |
| 4 | Quiz interativo | Pendente |
| 4 | Sistema de recompensas básico | Pendente |
| 5 | Painel do responsável (básico) | Pendente |
| 5-6 | Testes com 5-10 crianças | Pendente |
| 6 | Ajustes baseados em feedback | Pendente |

### Critérios de Sucesso
- [ ] Criança completa pelo menos 3 aulas
- [ ] Criança pede para continuar
- [ ] Pais demonstram interesse em pagar
- [ ] Taxa de conclusão > 60%
- [ ] Sessão média > 5 minutos

### Conteúdo MVP - Tema: Animais

| Aula | Conteúdo | Palavras |
|------|----------|----------|
| 1 | Animais da fazenda | perro, gato, vaca, caballo, pollo |
| 2 | Animais selvagens | león, elefante, mono, jirafa, tigre |
| 3 | Animais do mar | pez, ballena, delfín, tiburón, tortuga |
| 4 | Insetos e pequenos | mariposa, abeja, hormiga, araña, caracol |
| 5 | Revisão e jogo final | Todos os anteriores |

## Fase 2: Validação (Mês 2-3)

### Objetivo
Validar product-market fit com primeiros usuários pagantes.

### Entregas

| Item | Descrição |
|------|-----------|
| Landing page | Página de conversão com proposta de valor |
| Onboarding | Fluxo de cadastro simplificado |
| Pagamento | Integração com gateway (Stripe/PagSeguro) |
| Tema 2 | Novo tema: Cores e Números |
| Tema 3 | Novo tema: Família |
| Analytics | Implementação de métricas |
| Depoimentos | Coleta de feedback estruturado |

### Modelo de Monetização (Teste)

| Plano | Preço | Inclui |
|-------|-------|--------|
| Gratuito | R$ 0 | 2 primeiras aulas |
| Mensal | R$ 29,90 | Acesso completo |
| Anual | R$ 249,90 | Acesso completo + bônus |

### Critérios de Sucesso
- [ ] 100 usuários cadastrados
- [ ] 20 assinantes pagos
- [ ] NPS > 50
- [ ] Churn mensal < 10%

## Fase 3: Escala B2C (Mês 3-4)

### Objetivo
Expandir base de usuários e conteúdo.

### Entregas

| Item | Descrição |
|------|-----------|
| Nível 2: Conectar | Novo nível com frases e histórias |
| Temas 4-6 | Comida, Casa, Escola |
| Rotina sugerida | Feature de 15min/3x semana |
| Certificados | Certificados de conclusão por nível |
| App mobile | PWA otimizado para instalação |
| Relatórios avançados | Painel completo para pais |

### Marketing

| Canal | Ação |
|-------|------|
| Orgânico | Conteúdo em redes sociais (reels, stories) |
| Influencers | Parceria com mães influenciadoras |
| SEO | Blog com conteúdo educacional |
| Referral | Programa "Indique e ganhe" |

### Critérios de Sucesso
- [ ] 500 usuários ativos mensais
- [ ] 100 assinantes pagos
- [ ] CAC < R$ 50
- [ ] LTV > R$ 200

---

## Fase 4: Expansão B2B (Mês 4-6)

### Objetivo
Entrar no mercado escolar com proposta piloto.

### Entregas

| Item | Descrição |
|------|-----------|
| Versão escolas | Ambiente multi-aluno |
| Painel professor | Dashboard de turma |
| Alinhamento BNCC | Documentação curricular |
| Material apoio | Guia para professores |
| Licenciamento | Sistema de licenças por aluno |
| Piloto | 3-5 escolas piloto gratuitas |

### Modelo B2B

| Plano | Preço | Inclui |
|-------|-------|--------|
| Piloto | Gratuito | 1 turma, 3 meses |
| Escola Básica | R$ 15/aluno/ano | Até 100 alunos |
| Escola Premium | R$ 12/aluno/ano | 100+ alunos + suporte |

### Critérios de Sucesso
- [ ] 5 escolas em piloto
- [ ] 2 contratos fechados
- [ ] NPS escolar > 60
- [ ] Renovação > 70%

---

## Fase 5: Consolidação (Mês 6-12)

### Objetivo

### Entregas

| Item | Descrição |
|------|-----------|

| Novos idiomas | Inglês (replicar modelo) |
| Time comercial | Vendas B2B estruturadas |
| Investimento | Rodada seed |

### Metas

| Métrica | Meta |
|---------|------|
| Usuários ativos | 5.000 |
| Escolas | 20 |
| ARR | R$ 500.000 |
| Team | 5-8 pessoas |

## Riscos e Mitigações

| Risco | Probabilidade | Impacto | Mitigação |
|-------|---------------|---------|-----------|
| Baixa retenção | Média | Alto | Foco em narrativa e novidade constante |
| CAC muito alto | Média | Alto | Priorizar orgânico e referral |
| Desenvolvimento lento | Baixa | Médio | MVP mínimo, iterar rápido |
| Concorrente forte entra | Baixa | Alto | Diferenciação por localização |
| Escolas não adotam | Média | Médio | Piloto gratuito convincente |

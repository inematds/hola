# Problema de Mercado

## Problemas Identificados

### 1. Plataformas Genéricas
As plataformas existentes não são pensadas para crianças - são adaptações de conteúdo adulto com interface colorida.

**Impacto:** Baixo engajamento, frustração, desistência

### 2. Aprendizado Fragmentado
Vocabulário solto, sem contexto narrativo, sem progressão clara.

**Impacto:** Crianças não retêm o aprendizado, não conseguem usar o idioma

### 3. Falta de Prova de Progresso
Pais não conseguem ver claramente o que o filho está aprendendo.

**Impacto:** Cancelamento de assinatura, desconfiança no produto

### 4. Baixa Retenção Infantil
Crianças enjoam rápido de apps educacionais monótonos.

**Impacto:** Churn alto, CAC não se paga

### 5. Métodos Frios e Pouco Afetivos
Foco excessivo em gamificação, sem construção de vínculo emocional.

**Impacto:** Aprendizado superficial, sem motivação intrínseca

### 6. Custo Alto de Aulas ao Vivo
Aulas particulares ou em escolas de idiomas são caras para a maioria das famílias.

**Impacto:** Exclusão de grande parte do mercado potencial

### 7. Riscos de Segurança
Plataformas abertas com chat, anúncios ou conteúdo não moderado.

**Impacto:** Pais relutam em deixar crianças usarem sozinhas

## Oportunidade

Criar um produto que resolve todos esses problemas simultaneamente:

| Problema | Solução Hola |
|----------|--------------|
| Plataformas genéricas | Produto 100% infantil desde a concepção |
| Aprendizado fragmentado | Trilhas com narrativa e progressão |
| Falta de prova | Relatórios simples para pais |
| Baixa retenção | Personagem guia + histórias contínuas |
| Método frio | Afeto, repetição previsível, celebração |
| Custo alto | Assinatura acessível (fração do presencial) |
| Segurança | Ambiente fechado, sem chat, sem anúncios |

## Validação do Problema

### Pesquisa de Mercado
- Mercado EdTech Brasil: US$ 5,41 bilhões (2024)
- Crescimento projetado: 11,7% ao ano
- 373 startups focadas em K-12 no Brasil

### Gap Identificado
Nenhum concorrente domina o nicho específico de **espanhol para crianças brasileiras** com foco em metodologia infantil nativa.

# Hola - Plataforma Infantil de Espanhol

Uma plataforma educacional inovadora para ensino de espanhol para crianças de 6 a 12 anos, focada em aprendizado lúdico, progressivo e seguro.

## Visão do Produto

> **A criança precisa amar usar. O adulto precisa confiar para pagar.**

O Hola é uma plataforma 100% infantil (não adaptação de adulto), guiada por personagens e histórias, com progressão clara em níveis e relatórios simples para pais.

## Público-Alvo

- **Usuário final:** Crianças de 6 a 12 anos
- **Comprador:** Pais, responsáveis, escolas e projetos educacionais

## Tech Stack

- **Frontend:** React 18 + TypeScript
- **Styling:** Tailwind CSS
- **Audio:** Howler.js
- **Animações:** Framer Motion
- **Backend:** Node.js + Express (futuro)
- **Database:** PostgreSQL (futuro)
- **Auth:** Token por link (MVP)

## Começando

### Pré-requisitos

- Node.js 18+
- npm ou yarn

### Instalação

```bash
# Clonar repositório
git clone git@github.com:inematds/hola.git
cd hola

# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```

## Diferenciais Competitivos

- Produto 100% infantil
- Método pedagógico claro
- Experiência guiada (não solta)
- IA invisível (apoio, não protagonista)
- Relatórios para pais
- Afeto, narrativa e repetição
- Simplicidade operacional
- Alinhado à BNCC

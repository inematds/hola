O que é

Pensado para crianças desde a primeira tela

Diferente de plataformas que adaptam conteúdo adulto, o Hola foi projetado para a forma como crianças aprendem: narrativa, afeto e repetição, com personagem guia e currículo alinhado à BNCC.

🐾 Narrativa + afeto

Personagem guia, histórias e celebração de conquistas. Aulas curtas de cerca de 5 minutos.

🔒 Ambiente seguro

Sem chat, sem anúncios. Espaço fechado, pensado para a segurança das crianças.

📋 Prova para os pais

Relatórios simples e visuais de progresso, para o adulto confiar e acompanhar.

Como funciona

Três níveis progressivos

O produto é organizado em níveis que vão da familiaridade ao uso do idioma, com trilhas de aprendizagem em vez de vocabulário solto.

Nível 1 · Explorar (MVP)→ Nível 2 · Conectar→ Nível 3 · Dominar
Pré-requisitos

O que você precisa para rodar

App React + TypeScript com Vite e Tailwind. Leve de subir localmente.

Node.js + npm

Node 18+ recomendado, com npm para instalar dependências.

# verificar versão
node -v

Repositório

Clone o projeto e entre na pasta.

git clone https://github.com/inematds/hola.git
cd hola

Dependências

React Router, Framer Motion, Zustand e Tailwind já vêm configurados.

npm install
Guia de uso · passo a passo

Rodando o Hola localmente

Scripts reais do package.json: dev, build, preview, lint e test.

1

Instalar dependências

Clone o repositório e instale os pacotes.

git clone https://github.com/inematds/hola.git
cd hola && npm install
2

Rodar em desenvolvimento

Sobe o servidor Vite com hot reload.

npm run dev  # http://localhost:5173
3

Lint

Checagem de qualidade do código em src com zero warnings.

npm run lint  # ESLint em src (ts, tsx)
4

Testes

Executa a suíte de testes com Vitest.

npm run test  # Vitest
5

Build e preview

Compila tipos e gera o build de produção, depois pré-visualiza.

npm run build    # tsc && vite build
npm run preview  # serve o build localmente
Exemplos

Como o aprendizado acontece

Trilhas com progressão clara, em vez de palavras soltas.

🐶 Tema: animais

No Nível 1 (Explorar), a criança conhece sons e palavras básicas em espanhol por meio de um tema e um personagem guia.

🎯 Aulas de 5 minutos

Sessões curtas e narrativas mantêm a atenção infantil e elevam a retenção dia após dia.

👨‍👩‍👧 Painel dos pais

Relatórios visuais mostram o que o filho aprendeu, dando ao adulto a prova de progresso.

🏫 Escolas (B2B)

Espanhol alinhado à BNCC com relatórios de turma para escolas particulares de Ensino Fundamental.

Roadmap

Evolução do produto

Três níveis planejados, começando pelo MVP Explorar.

Nível 1
Explorar (MVP)Familiaridade com o espanhol: sons, palavras básicas, jogos simples, 1 tema e 1 personagem.
Nível 2
ConectarCompreensão e uso inicial: frases curtas, comandos e histórias guiadas, com novos temas mensais.
Nível 3
DominarUso mais autônomo do idioma e expansão do currículo, incluindo a frente B2B para escolas.
